# Music-school
Site institucional completo, moderno e emocionalmente envolvente para a Escola de Música Feitosa, inspirado nas melhores práticas de design de startups e focado no impacto social e na cultura nordestina.

O código foi desenvolvido em um único arquivo HTML contendo a estrutura, os estilos (via Tailwind CSS configurado com as cores da sua marca), as fontes (Poppins e Inter) e a interatividade (animações de rolagem, menu responsivo, contadores numéricos e uma simulação visual perfeita da integração com o Instagram).
